# GraphQL subscriptions sample(Apollo 3.3 and Actioncable 6.1)

## Client

@apollo/client@^3.3.12

## Server

rails (6.1.3)

## Setup

### database

```
$ docker-compose run api rails db:setup
```

### application

```
$ docker-compose up
```

## How to use

### client

http://localhost:3000/

### graphiql

http://localhost:4000/graphiql
